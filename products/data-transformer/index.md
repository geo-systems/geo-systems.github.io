## Supported Formats

<strong>Data Transformer</strong> supports the following data formats:
- [Comma-Separated Values (**CSV**)](https://en.wikipedia.org/wiki/Comma-separated_values) and all of its variations (e.g. TSV);
- [JavaScript Object Notation (**JSON**)](https://www.w3schools.com/whatis/whatis_json.asp);
- [Extensible Markup Language (**XML**)](https://www.w3schools.com/xml/xml_whatis.asp);
- [**YAML**](https://en.wikipedia.org/wiki/YAML);
- **Excel-compatible tabular workflows**;
- **SQL** with creation and insertion. SQL scripts can be created from CSV, JSON, XML, or YAML.
- More formats are to come!

With <strong>Data Transformer</strong> you can open local files or paste input data directly into the app.

## Frequently Asked Questions

**Is Data Transformer an offline app?**
Yes. Data Transformer is built for offline desktop use, so your data is processed locally on your machine.

**What formats can I convert between?**
You can convert between CSV, JSON, XML, YAML, and Excel-oriented tabular data workflows, and generate SQL creation/insertion scripts.

**Does Data Transformer upload my files to the cloud?**
No. The app is designed for local processing and does not require cloud uploads to run conversions.

**Is Data Transformer available on macOS and Windows?**
Yes. Data Transformer is available through the Apple App Store and Microsoft Store.

**Where can I download Data Transformer?**
You can download Data Transformer from the Apple App Store for macOS and from the Microsoft Store for Windows.
