<strong>TextQ</strong> is a macOS app for analysing and processing tabular data (such as CSV files) with a visual query builder or SQL.

Do you have CSV files that are too large for spreadsheet tools like Excel?
<strong>TextQ</strong> is built for that use case, and it allows you to:
- Load and preview the file;
- Rename, reorder, and hide columns;
- Change column formats (e.g. from text to number or date);
- Parse dates and numbers in many formats, languages, and locales;
- Filter rows using convenient UI query builder;
- Query the data via SQL (including "group by" and other advanced queries);
- Convert the UI builder's query to SQL;
- Index columns for faster query (useful for huge files);
- Export the result to CSV so it can be loaded by other apps (e.g. MS Excel).

Unlike cloud-first tools, <strong>TextQ</strong> processes your data locally on your computer.
On macOS, this is reinforced by Apple's sandbox model, and you can review app permissions before installation.
You can rest assured that your data stays on your machine.

## Why TextQ?

- Process large tabular datasets without uploading data to cloud tools.
- Switch between visual filtering and SQL depending on the task.
- Normalize date and number formats across mixed-locale source files.
- Export clean, query-ready outputs to CSV for downstream analysis.

TextQ is a practical alternative when spreadsheet apps become slow with large CSV analysis on macOS.

## Frequently Asked Questions

**Is TextQ an offline app?**
Yes. TextQ runs locally on your Mac and processes your CSV data on-device.

**Can TextQ handle CSV files that are too large for Excel?**
Yes. TextQ is designed for large CSV and tabular files where spreadsheet tools become slow or unreliable.

**Do I need to know SQL to use TextQ?**
No. You can use the visual query builder, and you can also use SQL when you need advanced filtering, grouping, or aggregations.

**What can I export from TextQ?**
You can export processed query results to CSV for use in other tools and workflows.

**Where can I download TextQ?**
You can download TextQ from the Apple App Store.
